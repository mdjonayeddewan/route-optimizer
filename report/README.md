# LaTeX Report

This folder contains the LaTeX source for the project report.

## Files

- `main.tex`: standalone report entrypoint inside this folder.
- `body.tex`: shared report body used by both the root-level and folder-level entrypoints.
- `references.bib`: supporting bibliography file.
- `figures/`: place screenshots or generated figures here if you want to add them later.

## Build

Compile `main.tex` from this folder or `Project_Report.tex` from the project root with a LaTeX editor or `pdflatex`.
